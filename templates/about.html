About This Project

Superstore Sales Prediction is a modern web application that showcases a complete machine learning workflow for retail sales forecasting. The project integrates data science best practices, robust MLOps, and an intuitive dashboard to deliver actionable business insights.

  • Real-time Prediction: Instantly forecast sales using a trained machine learning model built on real Superstore data.
  • Interactive Dashboard: Explore sales, profit, and trends by segment, category, region, and more, with dynamic visualizations and KPIs.
  • MLOps & Monitoring: Automated retraining, MLflow experiment tracking, and logging for continuous improvement and transparency.
  • Technical Stack: Python, Flask, scikit-learn, pandas, MLflow, Bootstrap, and modern data visualization libraries.

Business Value: This solution empowers retail teams to make data-driven decisions for inventory, pricing, and strategy by providing accurate forecasts and clear, actionable analytics.
Future Work: Planned enhancements include cloud deployment, advanced analytics (e.g., anomaly detection), and more customizable reporting.
